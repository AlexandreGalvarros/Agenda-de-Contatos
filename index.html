Agenda de Contatos



O código de a gendamento de de contatos tem como o objetivo demosntrar a evolução do código com relação ao passar das estapas, o qual é adicionado novas funcionalidades ao código anterior. Cujo o usuário pode se registrar e consultar seu cadastro em uma lista de contatos.



Tecnologias Usadas



C

Uso e Funcionalidade



Já dentro da IDE que desejares usar, como VS-CODE e GDB online, é necessário escolher com umas das opções dadas pelo o sistema para que siga com programa.Que são entre fazer cadastro, consultar se existe um cadastro, ver uma lista de todos os cadastros já registrados e sair.

Ao continuar com umas das opções o programa vai continuar normalmente. Como no caso de fazer um novo cadastro, adicionar suas informções como nome, e-mail e telefone para contato.

Já sobre sua funcionalidade, é bem simples e ilustrativo, pois demostra de uma forma bem análoga, como as é por trás de uma tela de cadastro das grandes big-techs funciona.
